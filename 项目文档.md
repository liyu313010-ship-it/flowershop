# 欢雨鲜花店项目文档

## 1. 项目概述

欢雨鲜花店是一个基于前后端分离架构的电商平台，主要功能包括商品管理、用户管理、订单管理、购物车管理、评价管理等。

## 2. 技术栈

### 2.1 后端技术栈
- **框架**: ASP.NET Core 9.0
- **数据库**: MySQL
- **ORM**: Entity Framework Core
- **认证**: JWT
- **API文档**: Swagger
- **验证**: FluentValidation
- **缓存**: 内存缓存
- **响应压缩**: Brotli + Gzip

### 2.2 前端技术栈
- **框架**: Vue 3
- **构建工具**: Vite
- **状态管理**: Pinia
- **路由**: Vue Router
- **HTTP客户端**: Axios
- **UI组件库**: Element Plus
- **样式**: Tailwind CSS

## 5. API接口文档

### 5.1 认证相关接口
- `POST /api/Auth/register` - 用户注册
- `POST /api/Auth/login` - 用户登录
- `POST /api/Auth/logout` - 用户登出

### 5.2 商品相关接口
- `GET /api/Products` - 获取商品列表
- `GET /api/Products/{id}` - 获取商品详情
- `GET /api/Products/featured` - 获取推荐商品
- `GET /api/Products/home` - 获取首页商品
- `GET /api/Products/category/{categoryId}` - 根据分类获取商品
- `GET /api/Products/search` - 搜索商品

### 5.3 分类相关接口
- `GET /api/Categories` - 获取分类列表

### 5.4 购物车相关接口
- `GET /api/Carts` - 获取购物车
- `POST /api/Carts` - 添加商品到购物车
- `PUT /api/Carts` - 更新购物车商品数量
- `DELETE /api/Carts/{productId}` - 从购物车删除商品

### 5.5 订单相关接口
- `GET /api/Orders` - 获取订单列表
- `GET /api/Orders/{id}` - 获取订单详情
- `POST /api/Orders` - 创建订单
- `PUT /api/Orders/{id}` - 更新订单状态

### 5.6 评价相关接口
- `GET /api/ProductReview/all` - 获取所有评价
- `GET /api/ProductReview/product/{productId}` - 获取商品评价
- `POST /api/ProductReview` - 创建评价
- `DELETE /api/ProductReview/{reviewId}` - 删除评价

## 7. 部署说明

### 7.1 后端部署
1. 配置 `appsettings.json` 中的数据库连接字符串和JWT密钥
2. 运行 `dotnet build` 构建项目
3. 运行 `dotnet run` 启动服务

### 7.2 前端部署
1. 配置 `.env` 文件中的API地址
2. 运行 `npm install` 安装依赖
3. 运行 `npm run build` 构建项目
4. 部署 `dist` 目录到Web服务器

## 8. 开发说明
